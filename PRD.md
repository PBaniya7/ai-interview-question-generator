# AI Interview Question Generator

## Problem

Candidates spend significant time searching for relevant interview questions and often prepare inefficiently.

## Goal

Generate personalized interview questions based on role, company, and experience.

## Target Users

- Product Managers
- Software Engineers
- Analysts
- Students

## MVP Features

- Role input
- Company input
- Experience input
- Interview question generation
- Preparation tips

## Success Metrics

- Number of generated sessions
- User satisfaction
- Time saved in preparation

## Future Enhancements

- Mock interviews
- Resume analysis
- Voice interview mode
- Personalized feedback
